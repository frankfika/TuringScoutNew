# Product Strategy

## 1. Product Thesis

AI projects increasingly need attention, GitHub credibility, agent usage, developer adoption, technical content, and community feedback. Many have budget, free credits, token/point systems, cloud coupons, early access slots, or bounty budgets, but they lack efficient distribution.

At the same time, users want to discover AI projects that are worth trying, can provide free credits, may have future airdrops, offer bounties, or help them build reputation.

AICookieDao connects these two sides with a lightweight discovery and task layer.

## 2. What It Is

AICookieDao is not initially a heavy community platform. It is a curated AI opportunity feed:

- ProductHunt-style AI project discovery
- CookieDAO-style campaign and contribution attribution
- Airdrop hunter-style reward discovery
- GitHub Trending-style open-source momentum
- Agent marketplace-style usage discovery

## 3. What It Is Not

- Not a fake GitHub star marketplace
- Not a pay-to-spam X platform
- Not a heavy DAO at launch
- Not a full bounty marketplace on day one
- Not a social network that requires every user to register first

## 4. Core User Groups

### AI Project Teams

Needs:

- Get early users and testers
- Grow GitHub stars naturally
- Generate technical content
- Get agent usage and feedback
- Activate Discord/community
- Promote free credits, points, grants, hackathons, waitlists, API trials
- Track campaign ROI without building in-house growth systems

### AI Opportunity Hunters

Needs:

- Find AI projects with rewards, credits, points, grants, or future upside
- Complete simple tasks quickly
- Avoid fake/low-quality projects
- Know which projects are trending
- Build a public AI builder profile over time

### Developers / Builders

Needs:

- Find useful AI tools early
- Earn bounties through real contributions
- Discover open-source projects worth starring, testing, integrating, or contributing to
- Build reputation through tutorials, issues, PRs, integrations, and benchmarks

### Technical Creators

Needs:

- Find sponsor-worthy AI projects
- Get campaign briefs and task suggestions
- Create useful content instead of random ads
- Show impact beyond views

## 5. Core Product Loop

1. Project submits an opportunity: launch, free credits, bounty, GitHub campaign, agent trial, points program, grant, hackathon, plugin challenge.
2. AICookieDao curates and publishes it as a task card or project page.
3. Users browse without login and click out to complete the task on the project’s own site/GitHub/Discord/X.
4. Users optionally connect GitHub/X/email only when they want to save, claim, or build reputation.
5. Project receives traffic, usage, GitHub attention, content, feedback, or contributions.
6. AICookieDao ranks projects by momentum, reward value, proof quality, and community interest.
7. More hunters and builders return daily for new opportunities.

## 6. The Lightweight Angle

The product should avoid forcing both sides into a heavy workflow.

For users:

- No login required to browse
- One-click outbound tasks
- Optional account only for watchlist, claim, profile, and leaderboard
- Telegram/Discord/email digest for repeat visits

For projects:

- Submit via a simple form
- No SDK required at first
- Campaign pages can be manually approved
- Basic analytics can be UTM-based
- Premium reporting can be handled manually for early customers

## 7. Differentiation

Compared with ProductHunt:

- Focused on AI tasks, rewards, usage, and open-source adoption
- More actionable than “upvote this launch”
- Better for ongoing campaigns, not just launch day

Compared with CookieDAO:

- Lighter, less token-dependent, less X-policy risk
- Rewards real product usage, GitHub activity, agent trials, and technical contributions
- Works for Web2 AI/open-source teams, not only crypto projects

Compared with airdrop sites:

- AI-native, not only crypto
- Includes free credits, API trials, grants, bounties, open-source tasks, agent testing, not only token speculation
- Higher quality filters and project credibility scoring

Compared with bounty platforms:

- More discovery-oriented
- Lower task friction
- Supports micro-tasks and non-code contributions
- Optimized for growth campaigns, not just engineering backlog

- “薅羊毛” for hook
- “真实使用” for quality
- “AI 项目早期机会” for aspiration
- “别刷假 star，赚真实贡献” for positioning

# 一页纸 PRD 草稿

> 归档说明：本文档是最早期的一页纸 PRD 草稿，已被 02-v1-prd.md 扩展替代。

---

**产品名**：TuringScout（暂定）

**一句话**：AI 行业的情报聚合平台 —— 自动追踪 GitHub、Hacker News 等平台的 AI 项目动态，帮助开发者、投资人、创业者发现机会。

**目标用户**：AI 开发者 & 技术投资人 & 科技从业者

**核心问题**：AI 行业变化太快，好项目的发现是分散的，缺少统一的 AI 项目情报工具。

**解决方案**：
- 自动聚合多平台 AI 项目数据
- 算法评分 + 人工补充保证质量
- 开放透明（评分规则公开、数据来源可查）

**V1 范围**：
- 数据自动抓取（GitHub Trending、HN）
- 项目排行榜（分类筛选、排序）
- 项目详情页
- Scout 提交系统

**V1 不做**：
- 不做代币
- 不做社区
- 不做付费
- 不做 API

**成功指标**：
- 100+ 项目收录
- 500 DAU（30天内）

**商业模式（远期）**：
- 代币经济（主要）
- 企业 API（次要）
- 广告合作（补充）

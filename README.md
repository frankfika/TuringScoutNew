# TuringScout

**AI 行业的 CookieDAO** — 追踪 GitHub、Twitter、知乎、Reddit、ArXiv、Hacker News 等平台的 AI 开源项目和技术趋势，为 AI 从业者、投资人、创业者提供实时情报和趋势分析。

## 快速入门

TuringScout 是一个 AI 项目发现和趋势分析平台，帮助你：

- 🔍 发现值得关注的优质 AI 开源项目
- 📈 追踪技术栈趋势和框架演进
- 🚀 发现早期爆发的 AI 项目机会
- 🎯 监控竞品和同类项目动态
- 🎁 通过贡献数据赚取积分（未来可兑换代币）

## 技术栈

- **前端**：Next.js + React + Tailwind CSS
- **后端**：Next.js API Routes
- **数据库**：PostgreSQL + Prisma ORM
- **部署**：Vercel / Cloudflare Pages

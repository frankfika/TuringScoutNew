## 分类启发

这些示例帮助建立了机会分类体系，详见 02-v1-prd.md 的"机会分类体系"章节。关键洞察：
1. 免费额度是最常见的发现需求
2. 开源项目发现是持续需求
3. Agent 试用是新兴机会类型
4. Bounty/竞赛有明确的价值

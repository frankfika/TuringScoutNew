# Go-To-Market Plan

## 1. Initial Wedge

Start as a daily AI opportunity feed, not a platform.

The first user promise:

> Come here every day to find AI projects worth trying for free credits, points, bounties, early access, or future upside.

The first project promise:

> Get listed in front of AI hunters, builders, creators, and early adopters without setting up a heavy campaign system.

## 2. Supply Acquisition

Curate opportunities from:

- AI project launch posts
- GitHub trending AI repos
- ProductHunt AI launches
- X posts from AI founders
- Discord community announcements
- Hugging Face projects
- Agent directories
- MCP server lists
- Hackathons and grants
- Cloud/API credit programs
- Web3 x AI points campaigns

Manual curation is enough at the beginning.

## 3. Demand Acquisition

Channels:

- X daily thread: “Today’s AI rewards”
- Telegram channel: fast updates
- Discord server: discussion and proof sharing
- Newsletter: weekly digest
- SEO pages: “best AI free credits”, “AI airdrops”, “AI agent bounties”, “open-source AI bounties”
- Reddit/HN only for high-quality reports, not spam
- Chinese channels: 即刻, 小红书, 知乎, 微信群, B站, 飞书群

## 4. Content Formats

Daily:

- 5 AI opportunities to try today
- New AI credits/points campaigns
- New GitHub AI projects worth watching

Weekly:

- AI rewards leaderboard
- Fastest-growing AI open-source projects
- Top agent trials
- Best bounties for builders
- New possible airdrops in AI

Monthly:

- AI project growth report
- Agent adoption report
- Open-source AI momentum report

## 5. Project Outreach Script

Short email/DM:

```
Hey {name}, we run AICookieDao, a lightweight discovery feed for AI users, builders, and reward hunters.

We feature AI projects offering free credits, early access, GitHub/open-source tasks, agent trials, bounties, grants, or points programs.

Your project looks like a good fit because {reason}. Want us to list your current campaign or help design a lightweight launch task?

We can start with a free listing and share click/interest data after it goes live.
```

## 6. Paid Pilot Offer

Offer a simple package first:

### AI Launch Boost

Includes:

- Featured listing for 7 days
- Campaign/task page
- Telegram/newsletter mention
- X thread inclusion
- Basic click report
- 3 suggested task formats

Price test:

- $299 early pilot
- $999 standard
- $2,500 with manual creator/builder outreach

### Developer Adoption Sprint

Includes:

- GitHub/project page optimization suggestions
- 2-3 builder tasks
- Featured placement
- Creator/developer outreach
- Weekly report

Price test:

- $2,000-$5,000 pilot

## 7. Community Strategy

Do not start with a generic Discord. Start with a Telegram or Discord alert channel:

- One-way announcement first
- Then add discussion after repeat users appear
- Recruit 20-50 power hunters/builders manually
- Give early members “Genesis Hunter” or “AI Builder Scout” badges

## 8. Viral Loops

User-side:

- Share task cards
- Weekly leaderboard images
- “I found this AI reward” referral links
- Public hunter profiles later

Project-side:

- “Featured on AICookieDao” badge
- Projects share their listing
- Weekly ranked projects share leaderboard

Creator-side:

- Creators use task feed for content ideas
- AICookieDao reposts best tutorials and demos

## 9. Risks

### Low-Quality Airdrop Vibe

Mitigation:

- Curate hard
- Add credibility and risk labels
- Mix easy rewards with high-quality builder tasks

### Platform Policy Risk

Mitigation:

- Avoid pay-to-spam
- Avoid guaranteed stars
- Use official project tasks
- Require sponsored disclosure for paid content

### Supply Not Enough

Mitigation:

- Broaden reward definition: free credits, waitlists, grants, hackathons, beta access, bounties, GitHub tasks, agent trials

### Users Do Not Register

This is acceptable. Monetization can start from project-side featured listings and reports. Registration should be optional.

## 10. Best First Markets

Chinese + global hybrid is attractive:

- Chinese users are familiar with 薅羊毛, 空投, 积分, 任务
- Global AI projects need low-cost user acquisition
- Chinese AI builders and creators need global AI opportunities
- Many China-origin AI projects want overseas GitHub/ProductHunt-style exposure

Suggested initial language:

- Bilingual UI later
- Chinese-first content can cold-start faster
- English project pages help global projects understand value

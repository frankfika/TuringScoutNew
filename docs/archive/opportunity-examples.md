# Opportunity Examples

These are example cards to guide curation and UI design. They are not live listings unless verified before publishing.

## Example 1: Free API Credits

- Title: Claim $20 AI API credits
- Type: Free Credits
- Difficulty: 1-Minute Click
- Reward: API credits
- CTA: Claim credits
- Project value: signup and first API call
- Proof: account created / API key activated
- Risk label: verify official link

## Example 2: Open-Source Agent Repo

- Title: Try a new browser agent and star only if useful
- Type: GitHub + Agent Trial
- Difficulty: 30-Minute Builder Task
- Reward: reputation badge / bounty if feedback accepted
- CTA: Run demo
- Project value: real usage, feedback, potential GitHub attention
- Proof: feedback form, issue, or demo screenshot
- Risk label: no guaranteed star

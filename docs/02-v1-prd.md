# 02 — V1 产品需求文档

## 一、V1 产品目标

**一句话目标**：上线一个完全由 AI 自动驱动的机会发现平台 —— AI 抓数据、AI 写点评、AI 打分，用户只看就行。

**核心差异化**：不是又一个 AI 工具目录，而是 **AI 帮你发现、理解、对比 AI 项目**。每条机会都有 AI 自动生成的深度点评。

**核心指标**：
- 100+ AI 机会收录，每条有 AI 自动点评
- 数据全自动更新（Tier 1 源每小时同步）
- 用户从打开页面到点击 CTA 的平均时间 < 30 秒
- 零用户参与：没有提交、没有注册、没有社区

**V1 不做**：用户提交、代币、创作者内容、个性化推荐、付费功能、API、用户注册登录、Scout 系统

---

## 二、目标用户

### 唯一画像：普通 AI 关注者

- **是谁**：对 AI 感兴趣的普通人，可能是开发者、学生、产品经理、或者单纯好奇
- **怎么用**：每天花 2 分钟扫一眼，看看有没有免费的好东西可以领
- **使用场景**：排队/通勤/午休时掏出手机刷一下，或者电脑上开个标签页偶尔刷新
- **核心需求**：快速发现"对我有用"的 AI 项目或福利，不需要学习怎么用这个平台

### 用户期望（按优先级）
1. 打开就能看，不需要注册
2. 一眼扫过去能找到感兴趣的东西
3. 点进去能看到 **AI 帮你分析好的深度信息**（好在哪、适合谁、和同类比怎么样）
4. 一键跳过去参与

## 三、核心设计原则

**列表页：5 秒扫完一屏**
- 每行只有：类型图标 + 项目名-机会标题 + 一句话描述 + 时间
- 没有标签堆叠，没有评分数字，没有用户信息

**详情页：AI 帮你理解**
- AI 自动生成的深度点评是详情页的核心内容
- 包括：亮点分析、适合人群、同类对比、多维度打分
- 人类管理员的角色是审核和修正 AI 的输出，不是从零创作

**数据来源：100% AI 自动化**
- 所有数据由 AI 从 GitHub/HN/HuggingFace 自动抓取
- 管理员审核 AI 抓取的内容质量后发布
- 不接受用户提交，保持数据质量可控

## 四、功能优先级

### P0（必须有，V1 上线必备）

| 功能 | 说明 |
|------|------|
| 机会列表（首页） | 按评分排序的机会列表，支持类型 Tab 筛选 |
| 机会详情 + AI 点评 | 好处、门槛、CTA，**AI 自动生成的深度点评**（亮点/适合人群/同类对比/打分） |
| 数据自动抓取 | GitHub Trending + HN + Hugging Face 定时同步 |
| AI 信息抽取 + 点评生成 | 从原始数据自动提取信息，**自动生成中文点评** |
| 评分排序 | 多维评分算法自动打分，AI 给出每个维度的解读 |
| 后台审核发布 | 管理员审核 AI 抓取+点评的内容，通过后发布 |

### P1（体验完整性）

| 功能 | 说明 |
|------|------|
| 排序切换 | 按评分 / 按最新 切换 |
| 项目详情页 | 点击项目名查看项目全貌和关联机会 |
| 基础后台 | 数据源管理、评分调整、手动抓取触发 |

### P2（锦上添花）

| 功能 | 说明 |
|------|------|
| 搜索 | 关键词搜索 |
| 趋势图表 | Star 增长曲线 |
| 邮件通知 | 新机会提醒（用户主动订阅） |
| 社交分享 | 一键分享链接 |

| 类型 | 图标 | 含义 | 示例 |

## 六、页面清单

| 页面 | 路由 | 核心内容 | 优先级 |
|------|------|---------|--------|
| 首页（机会列表） | `/` | Tab 筛选 + 排序 + 机会列表 | P0 |

| 后台面板 | `/admin` | AI抓取审核、数据源管理 | P0 |

## 七、数据来源

所有数据由 AI 自动抓取，不接受用户提交：

| 来源 | 频率 | 说明 |
|------|------|------|
| GitHub Trending | 每小时 | AI 相关 repos 自动筛选 |
| Hacker News /show | 每天 4 次 | Show HN AI 相关 |
| Hugging Face | 每天 4 次 | Trending models/spaces |

管理员在后台审核 AI 抓取并自动抽取的候选机会，通过后发布上线。

---

## 八、验收条件

**列表体验**：
- [ ] 打开首页，首屏能看到 5+ 条机会，无需滚动
- [ ] 每行信息不超过 4 个元素（图标、标题、描述、时间）
- [ ] Tab 切换响应 < 200ms
- [ ] 首页加载 < 2s

**详情体验**：
- [ ] AI 点评在首屏可见（亮点 + 适合人群 + 对比 + 打分解读）
- [ ] 每个评分维度有 AI 给出的一句解读，不是干巴巴的数字
- [ ] CTA 按钮点击后新窗口打开目标链接

**数据自动化**：
- [ ] GitHub Trending 每小时自动同步
- [ ] HN /show 每日自动同步
- [ ] AI 自动抽取项目名、描述、机会类型
- [ ] 自动去重（相同 URL 不重复入库）
- [ ] 数据源异常时告警

**后台管理**：
- [ ] 管理员可查看 AI 抓取的候选机会列表
- [ ] 可通过/拒绝候选机会（审核发布）
- [ ] 可查看数据源运行状态和手动触发抓取
- [ ] 可手动调整评分权重

**性能**：
- [ ] 页面加载 P95 < 2s
- [ ] API 响应 P95 < 500ms
- [ ] 移动端无横向滚动

# Task Taxonomy

## 1. Task Categories

### Free Credit Tasks

Examples:

- Sign up and claim API credits
- Join waitlist for compute credits
- Try a model playground
- Redeem partner coupon

User motivation: immediate value.
Project value: signups, activation, product trials.

### Points / Airdrop Speculation Tasks

Examples:

- Create account and complete onboarding
- Use agent for a certain number of sessions
- Invite friends
- Join Discord and verify
- Complete product quests

User motivation: future upside.
Project value: early community and engagement.

### GitHub Discovery Tasks

Examples:

- Visit repo and evaluate usefulness
- Star only if genuinely interested
- Fork and run demo
- Open a useful issue
- Submit a small PR
- Add project to a template or example

User motivation: reputation, bounty, learning.
Project value: authentic open-source growth.

Important: never sell “guaranteed stars”. Phrase as “discover, test, and support useful repos”.

### Agent Trial Tasks

Examples:

- Try an AI agent on a real task
- Submit success/failure report
- Compare two agents
- Record a workflow demo
- Connect agent to a tool or MCP server

User motivation: useful tools, rewards, creator content.
Project value: real usage, feedback, demos.

### Content Tasks

Examples:

- Write a tutorial
- Make a short demo video
- Publish a benchmark
- Write a comparison thread
- Translate docs
- Create a use-case guide

User motivation: cash, audience, reputation.
Project value: discoverability and education.

### Integration Tasks

Examples:

- Build MCP server
- Add LangChain/LlamaIndex example
- Create Cursor/VS Code setup guide
- Add Docker compose template
- Build Zapier/n8n workflow
- Create Hugging Face Space demo

User motivation: higher bounty, portfolio.
Project value: ecosystem adoption.

### Feedback Tasks

Examples:

- UX review
- Docs review
- Bug report
- Onboarding friction report
- Pricing feedback
- Competitor comparison

User motivation: easy contribution.
Project value: product improvement.

## 2. Task Difficulty Levels

### Level 1: 1-Minute Click

- Claim credits
- Join waitlist
- Follow project updates
- Visit GitHub repo
- Join Discord

### Level 2: 5-Minute Try

- Run demo in browser
- Use agent once
- Complete onboarding
- Submit quick feedback

### Level 3: 30-Minute Builder Task

- Run repo locally
- Write issue
- Test API
- Create simple demo
- Compare with competitor

### Level 4: 2-Hour Contributor Task

- Tutorial
- Video
- PR
- Integration
- Benchmark

### Level 5: Deep Campaign

- Full case study
- Enterprise pilot
- Complex integration
- Multi-agent workflow
- Long-form evaluation

- Early access
- Public reputation badge
- Featured creator placement

## 4. Ranking Formula

A simple initial ranking can use:

```
Opportunity Score = Reward Value + Project Credibility + Ease Score + Trend Score + Proof Quality - Risk Score
```

Where:

- Reward Value: cash/credits/points upside
- Project Credibility: team, funding, GitHub, usage, partners
- Ease Score: how fast users can complete it
- Trend Score: recent attention and growth
- Proof Quality: how verifiable the task is
- Risk Score: scam, spam, unclear reward, privacy risk, fake activity
